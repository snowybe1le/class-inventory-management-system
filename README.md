# Class & Inventory Management System

## Overview
A relational database system developed for Darul Niswah PPWK Demit to manage students, classes, payments, and inventory.

## Technologies
- Oracle APEX
- SQL

## Features
- Student registration and enrollment
- Payment tracking
- Class scheduling
- Inventory (materials & equipment)
- Supplier management

## Database Concepts
- ERD design
- Normalization (up to 3NF)
- Primary Key, Foreign Key, NOT NULL
- SQL queries (JOIN, aggregation, nested queries)

## Files
- schema.sql → database structure
- insert_data.sql → sample data
- queries.sql → SQL queries
